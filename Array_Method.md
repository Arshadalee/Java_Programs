# Java Array Methods:

## Array Length:

### array.length: 
Returns the number of elements in the array.

## array.clone()
Returns a shallow copy of the array.

## array.equals(Object obj)
Compares the specified object with the array for equality.

## array.toString()
Returns a string representation of the array, but for arrays of objects, this typically returns the memory address unless overridden.

## array.copyOf(int newLength)
Copies the array, truncating or padding with zero values to the specified new length.

## array.copyOfRange(int from, int to)
Copies a range of the array from the specified from index to the to index.

## array.sort()
Sorts the elements of the array into ascending order.

## array.fill(int value)
Fills the entire array with the specified value.

## array.binarySearch(int key)
Searches for the specified key in a sorted array using the binary search algorithm.

## array.equals(Object[] other)
Compares the specified array with another array for equality, considering element-wise comparison.
