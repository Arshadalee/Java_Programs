## 1. hashSet.size():
Returns the number of elements in the HashSet.

## 2. hashSet.isEmpty():
Returns true if the HashSet contains no elements.

## 3. hashSet.add(E e):
Adds the specified element to the HashSet if it is not already present.

## 4. hashSet.remove(Object o):
Removes the specified element from the HashSet if it is present.

## 5. hashSet.contains(Object o):
Returns true if the HashSet contains the specified element.

## 6. hashSet.clear():
Removes all elements from the HashSet.

## 7. hashSet.iterator():
Returns an iterator over the elements in the HashSet.

## 8. hashSet.addAll(Collection<? extends E> c):
Adds all of the elements in the specified collection to the HashSet if they're not already present.

## 9. hashSet.removeAll(Collection<?> c):
Removes from the HashSet all of its elements that are contained in the specified collection.

## 10. hashSet.containsAll(Collection<?> c):
Returns true if the HashSet contains all of the elements in the specified collection.

## 11. hashSet.equals(Object o):
Compares the specified object with the HashSet for equality.

## 12. hashSet.hashCode():
Returns the hash code value for the HashSet.

## 13. hashSet.forEach(Consumer<? super E> action):
Performs the given action for each element in the HashSet until all elements have been processed or the action throws an exception.

## 14. hashSet.stream():
Returns a sequential stream with the elements of the HashSet.

## 15. hashSet.parallelStream():
Returns a possibly parallel stream with the elements of the HashSet.
